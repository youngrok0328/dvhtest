# README #

회사 공통으로 사용할 VC 프로젝트 설정용 파일이 있습니다.

## IntekPlus.Default.v15.props ##

VC++ 2017 프로젝트에서 사용할 컴파일/링크 옵션에 대한 공통 설정 파일입니다.

## IntekPlus.Default.v16.props ##

VC++ 2019 프로젝트에서 사용할 컴파일/링크 옵션에 대한 공통 설정 파일입니다.

## IntekPlus.Default.v17.props ##

VC++ 2022 프로젝트에서 사용할 컴파일/링크 옵션에 대한 공통 설정 파일입니다.

## StandardEncoding.ps1 ##

소스 코드 파일의 인코딩을 UTF-8-BOM으로 일괄 변환하는 스크립트입니다.

## StandardFormatting.ps1 ##

파일 구조 표준이 적용된 h/cpp 파일에 대한 일괄 포매팅 스크립트입니다.

## BuildScript-ipis-vision.ps1 ##

(인텔 제외) iPIS Vision 솔루션을 빌드하기 위한 표준 스크립트입니다.
